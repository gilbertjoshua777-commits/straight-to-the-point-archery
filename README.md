# Straight to the Point Archery

## Project Overview
A responsive website created for Straight to the Point Archery, an archery instruction business. This project was developed as part of the University of Washington Front-End Web Development course.

## Features
- Responsive website design
- Home, Classes, Instructors, and Contact pages
- Navigation between website pages
- Image galleries and styling
- Font Awesome integration
- Accessibility features
- Anti-Discrimination Policy page

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git and GitHub
- GitHub Pages

## Viewing the Website
The website is deployed using GitHub Pages and can be viewed at: https://github.com/gilbertjoshua777-commits/straight-to-the-point-archery
